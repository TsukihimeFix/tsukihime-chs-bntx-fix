# 静态验证

## 验证边界

修复只改变受影响 BNTX 中每条纹理记录的两个有效元数据字节：

1. `flags`：`0x07` 或同包内的 `0x09` 改为 `0x01`
2. `layout`：`0x30-0x34` 或 `0x40-0x44` 改为对应的 `0x00-0x04`

BC7 image payload、宽高、格式、tile mode、swizzle、mip 数量和 image size 不变。
